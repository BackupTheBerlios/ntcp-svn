Peer-to-Peer Reliable Connectivity Across
Network Address Translator
A Software Approach to Solving NAT Traversal Problem

GABALLO Luca

Abstract

Peer-to-peer applications need direct connection between nodes, and Network Address Translation (NAT) cause well-knows difficulties since the peers have no globally valid IP address. Some NAT traversal approaches are emerging, but they suffer from several problems. This work analyzes the NAT functionalities, the UDP/TCP communication through the NATs, documents the different approaches and try to find a good solution for the NAT traversal problem.
