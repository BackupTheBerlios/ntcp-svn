NTCP: NAT traversal TCP
A Software Approach to Solving NAT Traversal Problem


Peer-to-peer applications need direct connection between nodes, and Network Address Translation (NAT) cause well-knows difficulties since the peers have no globally valid IP address. Some NAT traversal approaches are emerging, but they suffer from several problems. This work analyzes the NAT functionalities, the UDP/TCP communication through the NATs, documents the different approaches and supplies a solution for the NAT traversal problem.
